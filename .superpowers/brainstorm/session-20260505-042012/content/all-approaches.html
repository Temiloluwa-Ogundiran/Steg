Trade-offs

Approach A (Recommended)

  • Fastest mode switching (single click)
  • Fits template's "Mode Calibration Grid" perfectly
  • Cleanest horizontal scan

Approach B

  • Vertical pipeline metaphor is strong
  • Takes more width away from forms
  • Mode labels may truncate on small screens

Approach C

  • Zero switching friction
  • Overwhelming density with 3 file inputs

Overall

  • All three keep the terminal sidebar
  • All use the same color/type system
